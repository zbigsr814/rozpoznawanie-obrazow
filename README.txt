# rozpoznawanie-obrazow
 
Aplikacja C# w której zastosowano przetwarzanie obrazów. Możemy załadować dowolny obraz, na którym następnie możemy zrealizować takie funkcje jak filtr progowy, transformację kolorów, rozmycie obrazu, wyznaczenie osi symetrii objektów, wyznaczenie środków objektów. Program rozpoznaje czy figury na obrazie to koła, owale, kwadraty, czy prostokąty. Program wypisuje parametry rozpoznanych figur tj. promień, długość boków, odchylenie standardowe.
